# Solution

**Draw from the box labeled "Apples & Oranges".**

## Reasoning

Since every label is wrong, the box labeled "Apples & Oranges" contains **only apples OR only oranges** (not both).

**Case 1: You draw an apple**
- "Apples & Oranges" box contains only apples
- "Apples" box must contain oranges (can't be apples, label is wrong)
- "Oranges" box must contain apples & oranges (the remaining contents)

**Case 2: You draw an orange**
- "Apples & Oranges" box contains only oranges
- "Oranges" box must contain apples (can't be oranges, label is wrong)
- "Apples" box must contain apples & oranges (the remaining contents)

Either way, one draw reveals all three boxes' contents.
