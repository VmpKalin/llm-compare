# The Error

The error occurs in this step:

**"Divide both sides by (a-b): a + b = b"**

## Why It's Invalid

Since we started with **a = b**, we know that:
$$a - b = 0$$

**You cannot divide both sides of an equation by zero.** Division by zero is undefined in mathematics.

When you divide by (a-b), you're dividing by 0, which violates a fundamental rule of algebra. This is why the proof breaks down—all the subsequent steps are meaningless because they're based on an invalid operation.

---

**Key lesson:** Whenever you divide both sides of an equation by an expression, you must first verify that the expression is non-zero. If it could be zero (as it is here), you need to handle that case separately.
