To solve this problem, we need to determine if a given string is a palindrome while ignoring case, spaces, and punctuation. A palindrome reads the same forwards and backwards, so our task involves preprocessing the input string to remove non-alphanumeric characters and then checking if the resulting string is a palindrome.

### Approach
1. **Preprocess the Input String**: 
   - Convert all characters to lowercase to ensure case insensitivity.
   - Remove all non-alphanumeric characters (such as spaces, punctuation) from the string. This can be efficiently done using list comprehensions and the `isalnum()` method which checks if a character is alphanumeric.

2. **Check for Palindrome**:
   - After preprocessing, check if the resulting string reads the same forwards and backwards. This can be easily achieved by comparing the processed string with its reverse.

### Solution Code
```python
def is_palindrome(s):
    processed = ''.join([c.lower() for c in s if c.isalnum()])
    return processed == processed[::-1]
```

### Explanation
- **Preprocessing**: The input string `s` is converted into a new string `processed` where each character is checked using `isalnum()`. Only alphanumeric characters are included, and they are converted to lowercase.
- **Palindrome Check**: The processed string is then compared with its reverse. If they are equal, the function returns `True`, indicating that the input string is a palindrome; otherwise, it returns `False`.

This approach efficiently handles all edge cases, including empty strings and strings with only non-alphanumeric characters, ensuring correct results for all scenarios.
