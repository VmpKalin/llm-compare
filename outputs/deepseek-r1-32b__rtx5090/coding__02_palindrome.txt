To solve this problem, we need to determine if a given string is a palindrome while ignoring case, spaces, and punctuation. A palindrome reads the same forwards and backwards when non-alphanumeric characters are removed and the remaining characters are converted to lowercase.

### Approach
1. **Normalize the String**: Convert all characters in the string to lowercase and remove any non-alphanumeric characters.
2. **Check for Palindrome**: Compare the normalized string with its reverse to determine if it is a palindrome.

The steps involve processing each character of the input string, filtering out non-alphanumeric characters, converting the remaining characters to lowercase, and then checking if the resulting string reads the same forwards and backwards.

### Solution Code
```python
def is_palindrome(s):
    processed = [char.lower() for char in s if char.isalnum()]
    cleaned = ''.join(processed)
    return cleaned == cleaned[::-1]
```

### Explanation
- **Normalization**: The code processes each character of the input string `s`. It checks if each character is alphanumeric using `isalnum()` and converts it to lowercase. This results in a list of processed characters.
- **String Construction**: The processed characters are joined into a single string, which contains only lowercase alphanumeric characters from the original string.
- **Palindrome Check**: The cleaned string is compared with its reverse (using slicing `[::-1]`). If they are equal, the function returns `True`, indicating the input string is a palindrome; otherwise, it returns `False`.

This approach efficiently handles all edge cases, including empty strings and strings with mixed cases and punctuation.
