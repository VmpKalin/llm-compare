To solve this problem, we need to write a Python function `solve(n)` that generates a list of strings for numbers from 1 to n. The rules are as follows:
- Multiples of 3 should be replaced with "fizz".
- Multiples of 5 should be replaced with "buzz".
- Multiples of 7 should be replaced with "bang".
- If a number is a multiple of more than one of these, the corresponding strings should be concatenated in the order "fizz", "buzz", and then "bang".

### Approach
1. **Loop through each number**: Iterate from 1 to n (inclusive).
2. **Check divisibility**: For each number, check if it is a multiple of 3, 5, or 7 in that specific order.
3. **Build the result string**: If a number is a multiple of any of these numbers, append the corresponding string ("fizz", "buzz", or "bang") to the result string for that number.
4. **Handle non-multiples**: If none of the divisibility conditions are met, convert the number to a string and add it to the result list.

### Solution Code
```python
def solve(n):
    result = []
    for i in range(1, n + 1):
        s = ''
        if i % 3 == 0:
            s += 'fizz'
        if i % 5 == 0:
            s += 'buzz'
        if i % 7 == 0:
            s += 'bang'
        result.append(s if s else str(i))
    return result
```

### Explanation
- **Loop through each number**: The loop runs from 1 to n, ensuring we process each number in the range.
- **Check divisibility**: For each number, we check if it is divisible by 3, then by 5, and finally by 7. This ensures that the strings are concatenated in the correct order.
- **Build result string**: Depending on the divisibility checks, the corresponding strings ("fizz", "buzz", "bang") are appended to form the result for each number.
- **Handle non-multiples**: If no conditions are met (i.e., the result string is empty), we convert the number to a string and add it to the result list.

This approach efficiently handles all cases, ensuring that each number is processed correctly according to the given rules.
