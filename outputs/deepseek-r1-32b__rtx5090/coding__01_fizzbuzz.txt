To solve this problem, we need to write a Python function `solve(n)` that generates a list of strings for numbers from 1 to n. Each number in the list is transformed based on specific rules related to its divisibility by 3, 5, and 7.

### Approach
The approach involves iterating through each number from 1 to n and checking its divisibility by 3, 5, and 7 in that order. For each number:
1. If the number is divisible by 3, append "fizz" to a string.
2. If the number is divisible by 5, append "buzz" to the same string.
3. If the number is divisible by 7, append "bang" to the string.

If none of these conditions are met, the number itself is converted to a string and added to the result list. Otherwise, the concatenated string from the checks is added.

### Solution Code
```python
def solve(n):
    result = []
    for i in range(1, n + 1):
        s = ''
        if i % 3 == 0:
            s += 'fizz'
        if i % 5 == 0:
            s += 'buzz'
        if i % 7 == 0:
            s += 'bang'
        result.append(s if s else str(i))
    return result
```

### Explanation
- **Initialization**: We start by initializing an empty list `result` to store the transformed strings.
- **Loop through numbers**: For each number from 1 to n, we initialize an empty string `s`.
- **Check divisibility**:
  - If the number is divisible by 3, "fizz" is appended to `s`.
  - If the number is divisible by 5, "buzz" is appended to `s`.
  - If the number is divisible by 7, "bang" is appended to `s`.
- **Build result**: After checking all conditions, if `s` is non-empty, it is added to the result list. Otherwise, the string representation of the number itself is added.

This approach ensures that each number is checked in the specified order and transformed correctly according to the given rules.
