To solve this problem, we need to determine the second largest unique value in a given list of numbers. If there are fewer than two unique values, the function should return `None`.

### Approach
The approach involves the following steps:
1. **Remove Duplicates**: Convert the input list into a set to eliminate any duplicate values.
2. **Check for Sufficient Unique Values**: If the resulting set has fewer than two elements, return `None` since we cannot determine a second largest value in such cases.
3. **Sort and Find Second Largest**: Sort the unique values in ascending order and then return the second last element from this sorted list, which will be the second largest unique value.

### Solution Code
```python
def second_largest(nums):
    unique = list(set(nums))
    if len(unique) < 2:
        return None
    unique_sorted = sorted(unique)
    return unique_sorted[-2]
```

### Explanation
1. **Removing Duplicates**: By converting the input list to a set, we ensure that all duplicate values are removed, leaving only unique elements.
2. **Checking for Sufficient Unique Values**: If there are fewer than two unique values, it is impossible to determine a second largest value, so we return `None`.
3. **Sorting and Finding Second Largest**: After sorting the unique values in ascending order, the second last element (at index `-2`) will be the second largest unique value in the original list.

This approach efficiently handles duplicates and ensures that we correctly identify the second largest unique value when it exists.
