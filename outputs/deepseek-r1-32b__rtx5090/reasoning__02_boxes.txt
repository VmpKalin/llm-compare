To solve the puzzle, draw from the box labeled "Apples & Oranges". Since all labels are incorrect, this box must contain only apples or only oranges.

- **If you draw an apple**, the box contains only apples. The box labeled "Oranges" cannot have oranges (as it's mislabeled) and can't have apples either (since we've already identified the apples), so it must contain both apples and oranges. Finally, the box labeled "Apples" must contain only oranges.

- **If you draw an orange**, the box contains only oranges. The box labeled "Apples" cannot have apples or oranges (as it's mislabeled) and thus must contain both. Consequently, the box labeled "Oranges" must contain only apples.

By drawing from the mixed label box first, you can logically deduce the contents of all three boxes with just one fruit drawn.

**Answer:** Draw from the "Apples & Oranges" box. If you draw an apple, it contains only apples; then the "Oranges" box has both, and the "Apples" box has oranges. If you draw an orange, it contains only oranges; then the "Apples" box has both, and the "Oranges" box has apples.
