## The Flaw: Division by Zero

The error occurs in **Step 5: "Divide both sides by (a−b)"**.

### Why It's Invalid

The proof begins by stating **a = b**, which means:

$$a - b = 0$$

Dividing both sides by $(a-b)$ is therefore **dividing by zero**, which is undefined and not a valid algebraic operation.

### What's Actually Happening

At Step 4, the equation is legitimately true:

$$(a+b)(a-b) = b(a-b)$$

This is just $0 = 0$, since both sides contain the factor $(a-b) = 0$. The equation holds, but it carries **no information** about the relationship between $a$ and $b$ beyond what we already assumed.

When you "cancel" $(a-b)$ from both sides, you are performing:

$$\frac{(a+b)\cancel{(a-b)}}{\cancel{(a-b)}} = \frac{b\cancel{(a-b)}}{\cancel{(a-b)}}$$

which is the operation $\dfrac{0}{0}$, an **indeterminate form**. This step is not permitted under any circumstances in standard arithmetic or algebra.

### Summary

| Step | Valid? | Reason |
|------|--------|--------|
| a = b | ✓ | Given assumption |
| a² = ab | ✓ | Multiply both sides by a |
| a²−b² = ab−b² | ✓ | Subtract b² from both sides |
| (a+b)(a−b) = b(a−b) | ✓ | Correct factoring |
| **Divide by (a−b)** | **✗** | **(a−b) = 0; division by zero** |

Every step after that point is built on an invalid operation, making the conclusion $2 = 1$ meaningless.
